# Iris Flower Classification

## Project Overview
This project uses Machine Learning to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Accuracy
- 100%

## Features
- Data preprocessing
- Data visualization
- Model training
- Prediction
- Accuracy evaluation
- Confusion matrix visualization

## Conclusion
The model successfully classified iris flowers with high accuracy using supervised machine learning techniques.
